## Installation and Setup Instructions

Installation:

1. Clone the repo

```sh
git clone https://github.com/kitharvey/shtonks.git

```
2. Install NPM packages

```sh
yarn
```


To Start Server:

```sh 
yarn start
```



To Visit App:

```sh 
localhost:3000
```
